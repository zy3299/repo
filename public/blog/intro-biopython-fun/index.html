How many cuts total in the sequence do you expect if you digest with both of these restriction enzymes? How many fragments do you expect?

import re
re.findall(r"A.TAAT|GC[^CT][^CG]TG",string)
## ['GCGTTG', 'ATTAAT', 'GCAATG', 'ACTAAT']

Four cuts in total so that five fragments would be expected.

Assume the restriction enzymes cut the sequence (string, above) at the midpoint of the binding site , so ANTand GCR. Using re.split(), cut the sequence at the cut points to digest the sequence, yielding the correct fragments. You might find that two separate re.split() commands is the easier way to go, but this my require a loop. Once you have the correct fragments, then use the count_chars() function from class to count the number of each base in each fragment. Modify it to additionally report the total length of each fragment.

#split by restriction enzymes
stringa=re.sub(r"(A.TAAT|GC[^CT][^CG]TG)",r"\1@",string)
stringb=re.sub(r"ATG@","@ATG",stringa)
stringc=re.sub(r"TTG@","@TTG",stringb)
stringd=re.sub(r"AAT@","@AAT",stringc)
string_frag=re.split("@",stringd)
print(string_frag)

def count_chars(sentence): 
     counts = {} 
     for c in sentence: 
         if c in counts: 
            counts[c]+=1 
         else:
             counts[c]=1 
     for c in counts:
         print(c, "appears", counts[c],"times")
